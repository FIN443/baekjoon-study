cmd(⌘)+return(⏎) -> 아래에 줄 삽입
cmd(⌘)+shift(⇧)+return(⏎) -> 위에 줄 추가
cmd(⌘)+shift(⇧)+K -> 현재 줄 삭제
cmd(⌘)+D -> 단어 여러개 선택
opt(⌥)+cmd(⌘)+(↑,↓) -> 방향키 방향으로 커서복제(멀티커서)
shift(⇧)+opt(⌥)+(↑,↓) -> 방향키 방향으로 현재 줄 복사
cmd(⌘)+(←,→) -> 줄 끝으로 커서 이동
opt(⌥)+(←,→) -> 다음 공백(" ")까지 이동
shift(⇧)+opt(⌥)+(←,→) -> 다음 공백(" ")까지 선택
shift(⇧)+cmd(⌘)+(←,→) -> 줄 끝까지 선택
cmd(⌘)+I -> 자동완성 가능한거 목록 보기
shift(⇧)+cmd(⌘)+space(␣) -> 함수(or 클래스)에 대한 설명보기
